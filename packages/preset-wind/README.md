# @unocss/preset-wind

Tailwind / Windi CSS compact preset for UnoCSS, [UnoCSS](https://github.com/antfu/unocss).

## Installation

```bash
npm i -D @unocss/preset-wind
```

```ts
import presetWind from '@unocss/preset-wind'

Unocss({
  presets: [
    presetWind()
  ]
})
```
